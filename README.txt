README

Per utilizzare questa applicazione è necessario utilizzare il DBMS Postgres.

E' necessario inizializzare il database con il codice sql di init allegato dopo aver creato un nuovo database (init.sql)

Ad ogni avvio del programma sono necessarie le informazioni di accesso al DBMS:
	- Nome database
	- Porta database
	- Nome utente
	- Password

E' possibile utilizzare l'archivio jar già compilato (da terminale java -jar Esame.jar)

Nella cartella /src è presente il codice sorgente suddiviso nei suoi package.
